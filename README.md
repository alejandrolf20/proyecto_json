# proyecto_json  Vamos a extraer información del archivo JSON de la API de Pokemon. En particular, vamos a extraer información del primer Pokemon, que tiene un ID de 1. Las funciones que vamos a crear son:

1.-Una función que liste el nombre, el peso y la altura del Pokemon.
2.-Una función que muestre cuántos tipos de Pokemon hay.
3.-Una función que permita al usuario buscar un movimiento del Pokemon y mostrar información sobre ese movimiento, si está disponible.
4.-Una función que busque información sobre los tipos de Pokemon que son débiles contra el tipo del Pokemon que estamos buscando.
5.-Una función que muestre información sobre el Pokemon que está en la posición siguiente en la Pokédex.
